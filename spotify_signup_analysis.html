3. Description of the System

Spotify is a digital music streaming platform that allows users to listen to music, podcasts, and curated playlists. Users are prompted to enter data when they create an account through the sign-up page. The purpose of the sign-up process is to create a personalized account that enables users to save playlists and receive customized recommendations.

4. Data Collected

Visible form fields:

  • Email address
  • Password
  • Display name/username
  • Date of birth
  • Gender (optional)

Inferred or automatically collected data:

  • IP address
  • Device type and operating system
  • Browser type
  • Location (approximate)

Behavioural data collected after sign-up:

  • Songs listened to
  • Search queries
  • Playlist creation
  • Time spent listening
  • Interaction with ads or recommendations

6. Ethical Analysis

One ethical concern is user privacy, as Spotify collects detailed behavioural data that can reveal personal preferences, moods, or routines. Another concern is data retention, since long-term storage of listening history increases the risk of harm if the data is breached. If this data were mishandled, users could face profiling and targeted manipulation through advertising.

7. Recommendations

  • Reduce data collection by limiting the amount of behavioural data stored long-term.
  • Provide clearer and more accessible explanations of how user data is used and retained.
  • Strengthen security by making two-factor authentication mandatory for all accounts.
